Number Guessing Game:

Overview:

The game generates a random number within a specified range, and the user attempts to guess the number within a limited number of attempts.

Technologies Used:

Java & 
Java Standard Libraries (java.util.Scanner, java.util.Random)

How the Game Works:

The system generates a random number between 1 and 100.
The user has 5 attempts to guess the correct number.
After each guess, the program provides feedback:
Too low/
Too high/
Correct guess.
After each round, the user can choose to play again.
At the end of the game, the total rounds played and rounds won are displayed.
